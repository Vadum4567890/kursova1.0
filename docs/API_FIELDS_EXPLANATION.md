# Пояснення полів API відповідей

## Автомобіль (Car)

### Основні поля:

**`id`** (integer)
- Унікальний ідентифікатор автомобіля в базі даних
- Генерується автоматично при створенні

**`brand`** (string)
- Марка автомобіля
- Приклад: "BMW", "Toyota", "Mercedes"

**`model`** (string)
- Модель автомобіля
- Приклад: "X5", "Camry", "C-Class"

**`year`** (integer)
- Рік випуску автомобіля
- Використовується для розрахунку вартості прокату (новіші автомобілі дорожчі)

**`type`** (enum: "economy" | "business" | "premium")
- Тип автомобіля
- **economy** - економ клас (найдешевші)
- **business** - бізнес клас (середня ціна, +30% до базової)
- **premium** - преміум клас (найдорожчі, +100% до базової)

### Фінансові поля:

**`pricePerDay`** (decimal)
- Вартість прокату за один день
- Приклад: "1000.00" = 1000 грн/день
- Використовується для розрахунку загальної вартості угоди

**`deposit`** (decimal) ⭐ ВАЖЛИВЕ
- **ЗАСТАВНА СУМА** - гроші, які клієнт залишає перед отриманням автомобіля
- Повертається клієнту після успішного повернення автомобіля
- Якщо автомобіль повернуто в неналежному стані - з застави можуть утримати штраф
- Приклад: "0.00" означає, що застава не встановлена (потрібно встановити)

### Статус:

**`status`** (enum: "available" | "rented" | "maintenance")
- Поточний статус автомобіля
- **available** - доступний для прокату
- **rented** - зараз в прокаті
- **maintenance** - на обслуговуванні/ремонті

### Додаткова інформація:

**`description`** (string, optional)
- Опис автомобіля
- Додаткова інформація про стан, особливості тощо

**`imageUrl`** (string, optional)
- Посилання на фото автомобіля
- Використовується для відображення в каталозі

### Технічні поля:

**`createdAt`** (timestamp)
- Дата та час створення запису в системі

**`updatedAt`** (timestamp)
- Дата та час останнього оновлення запису

---

## Приклад використання deposit:

1. **Клієнт хоче взяти автомобіль в прокат:**
   - Автомобіль має `deposit: 5000.00` (5000 грн застава)
   - Клієнт сплачує заставу перед отриманням

2. **Під час прокату:**
   - Застава "заморожена" - клієнт не може її використати

3. **Після повернення:**
   - Якщо автомобіль в хорошому стані → застава повністю повертається
   - Якщо є пошкодження → з застави утримується штраф, решта повертається

---

## Рекомендації для вашого прикладу:

У вашому прикладі `deposit: "0.00"` - це означає, що застава не встановлена. 

**Рекомендується встановити заставу:**
- Для economy: 3000-5000 грн
- Для business: 5000-10000 грн  
- Для premium: 10000-20000 грн

**Приклад оновлення через API:**
```json
PATCH /api/cars/1
{
  "deposit": 5000.00
}
```
